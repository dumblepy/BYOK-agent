## Reading this inventory

- `runtime` identifies the package declared in `dependencies` and included in the extension build dependency graph.
- `development` identifies build, lint, test, packaging, and their transitive dependencies. They are not included in the VSIX runtime payload.
- Multiple versions of one package are intentionally listed as separate rows.
- SPDX expressions retain the package-declared alternatives; this inventory does not silently select one license.
- For packages whose license text is not copied into this repository, the linked source package or SPDX text is the authoritative notice for that package version.
